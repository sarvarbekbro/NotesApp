# 📝 Notes App

A simple and interactive **Notes App** built using **HTML, CSS, and JavaScript**.
This application allows users to create, edit, delete, and persist notes directly in the browser using local storage.

---

## 🚀 Features

* ➕ Create new notes instantly
* ✏️ Edit notes directly (contenteditable)
* 🗑️ Delete notes with one click
* 💾 Automatic saving using browser localStorage
* 🔄 Notes persist even after page refresh
* 🎯 Clean and minimal user interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the app
* **CSS3** – Styling and layout
* **JavaScript (Vanilla)** – Logic and interactivity
* **localStorage API** – Data persistence

---

## ⚙️ How It Works

1. User clicks the **"Create Note"** button
2. A new editable note is added to the screen
3. User can type and edit the note content
4. Notes are automatically saved in **localStorage**
5. On page reload, saved notes are restored
6. Clicking the delete icon removes the note
